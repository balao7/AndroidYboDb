h1. AndroidYboDb

Ne trouvant pas de librairie pour pouvoir utiliser un mapping objet sur android pour la base de donnée, j'ai développé ma propre librairie.
Cette librairie est sans prétention, elle m'a juste rendu service pour mes besoins, ce qui sera peut être votre cas aussi.
